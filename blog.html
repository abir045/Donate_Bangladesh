Important FAQ

What is the Document Object Model (DOM)?

The DOM represents a html document with a logical tree. Each branch of the tree ends in a node, and each node contains objects. DOM methods allow programmatic access to the tree. With them, you can change the document's structure, style, or content.

How do you select an element from the DOM

JavaScript provides several methods to select DOM elements. Here are some of them: getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for multiple child elements, rather than adding individual event listeners to each child. This approach takes advantage of event bubbling, where events triggered on a child element "bubble up" through the DOM hierarchy, eventually reaching the parent element. It minimizes the need to manage multiple event listeners in the DOM and optimizes performance. It also works well with dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using the DOM in several ways. The DOM provides various methods and properties for working with attributes such as getAttribute(attributeName), setAttribute(attributeName), removeAttribute(attributeName). You can manipulate an element's styles using style property and classList.
